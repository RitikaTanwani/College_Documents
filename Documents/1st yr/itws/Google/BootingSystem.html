What does it mean 'boot your computer'?

  • 3 years ago
  • Report Abuse

Best Answer - Chosen by Voters

When you initially start your computer, it loads a "block" (typically 512 bytes) of data off of the hard drive. This block, called a "boot block" then contains enough code to load the rest of the operating system. This code is also known as a "boot strap".

Some boots have a small strap at the back of them to help you put the boots on, by pulling on the strap. There is an old phrase about pulling one's self up by one's bootstraps (physically impossible), which means being determined and self reliant.

Loading an operating system was likened to pulling one's self up by one's boot straps (i.e. a seemingly impossible task), thus the process became known as "booting".

    This term goes back to the early days of computers. In the olden days (the 1970s), a computer just being turned on had no programs in it at all. It was completely -stupid-. Well, in order to load a program, you need a program that loads the program! So how do you load the program that loads the program to load programs? This problem is like lifting yourself up by your own bootstraps. The answer was to key in a short program, by hand. This program would read a longer program from paper tape, and -that- program was the program that loaded the operating system and other programs. That was called a bootstrap loader. The process came to be known as 'booting'.

    Back in the 1970s I spent some time working as a typesetter, using a PDP 11 that worked with an IBM Selectric typewriter as a terminal. This was ultramodern stuff in those days, but if you saw it today you'd laugh. To start up the PDP-11 you had to key in about 20 bytes, using rows of toggle switches in the front, that is, you entered it BIT BY BIT. Then you'd read a paper tape, then a longer paper tape, and then you were ready to go. This took about ten minutes. And several times during a shift the computer would get 'confused' and you had to go through this all again. We have sure come a long way in 30 years!
